Mac Modify the application's default language

用于 Mac app 无法自定义语言

Example:

Get App's Bundle Identifier

# 获取 BUndle ldentifier
➜  MacOS mdls -name kMDItemCFBundleIdentifier /Applications/OmniFocus.app
kMDItemCFBundleIdentifier = "com.omnigroup.OmniFocus3"

# 设置App使用语言
➜  MacOS defaults write com.omnigroup.OmniFocus3 AppleLanguages '("en")'

# 重置默认语言
➜  MacOS defaults delete com.omnigroup.OmniFocus3 AppleLanguages
