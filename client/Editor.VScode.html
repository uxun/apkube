Visual Studio Code

Mac 键盘符号说明

Tag Command
⏏ == 电源开关键 Home == Fn + ←
⎋ == Escape (Esc) End == Fn + →
⌘ == Command ↑ == 上箭头
⇧ == Shift ↓ == 下箭头
⇪ == Caps Lock ← == 左箭头
⌥ == Option → == 右箭头
⌃ == Control ⇞ == Page Up（Fn+↑）
↩ == Return/Enter ⇟ == Page Down（Fn+↓）
⌫ == Delete ⇤ == 左制表符（Shift+Tab）
⌦ == 向前删除键（Fn+Delete） ⇥ == 右制表符（Tab键）
