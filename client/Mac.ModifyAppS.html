Mac Modify the application's default language and other

用于 Mac app 无法自定义语言

更改系统语言sudo languagesetup

Example:

Get App's Bundle Identifier

# 获取 BUndle ldentifier
➜  MacOS mdls -name kMDItemCFBundleIdentifier /Applications/OmniFocus.app
kMDItemCFBundleIdentifier = "com.omnigroup.OmniFocus3"

# 设置App使用语言
➜  MacOS defaults write com.omnigroup.OmniFocus3 AppleLanguages '("en")'

# 重置默认语言
➜  MacOS defaults delete com.omnigroup.OmniFocus3 AppleLanguages

Others

右键菜单打开方式出现重复项的解决办法

/System/Library/Frameworks/CoreServices.framework/Versions/A/Frameworks/LaunchServices.framework/Versions/A/Support/lsregister -kill -r -domain local -domain user

macOS 中刷新 DNS 缓存

sudo killall -HUP mDNSResponder

安装「任何来源的软件」

sudo spctl --master-disable

缩短隐藏的 Dock 栏在鼠标靠近时弹出的时间

# set 
defaults write com.apple.dock autohide-delay -float 0; killall Dock
or
defaults write com.apple.dock autohide-time-modifier -float 0; killall Dock

# default
defaults write com.apple.dock autohide-delay -float 1; killall Dock
or
defaults delete com.apple.dock autohide-time-modifier; killall Dock

休眠状态下是否启用网络

其中 -b 是应用到使用电池时，-c 是充电状态下, -u 是使用 UPS，-a 是所有情况下。一般推荐使用 -b 的参数就可以，使用电池时 tcpkeepalive = 0

# set
sudo pmset -b tcpkeepalive 0
# default
sudo pmset -b tcpkeepalive 1

文件夹英文显示

终端中运行touch ~/Downloads/.localized，重启 Finder （按住 ⌥ 在 dock 的 Finder 上右键）修好了。 如果是「图片」文件夹，修复的命令就是 touch ~/Pictures/.localized  ，以此类推。
