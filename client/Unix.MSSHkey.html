Configure Multiple Key

生成特定key

ssh-keygen -t rsa -C '{name}@gmail.com' -f ~/.ssh/id_rsa_github

将key 添加ssh密钥链中, 否则重启会Forgot id_rsa_github

# 添加
ssh-add -K ~/.ssh/id_rsa_github
# 查看
ssh-add -l

创建本地配置文件~/.ssh/config

Optionally replace ~/.ssh/id_rsa with the path to your key. To add additional keys, add a new line for each key with: IdentityFile /path/to/your_key.

Host *
  UseKeychain yes
  AddKeysToAgent yes
  IdentityFile ~/.ssh/id_rsa_github
  IdentityFile ~/.ssh/id_rsa_xxx
  IdentityFile ~/.ssh/id_rsa_xxxx

Test

ssh -T git@{username}.github.com
