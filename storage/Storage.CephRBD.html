Ceph RBD

Ceph块设备也称为RBD或RADOS 块设备。

1.ENV Prepare

1.Ceph Storage Cluster is in an active + clean
2.Kernel Version > (4.19.z | 4.14.z)
3.不建议btrfs用于支持Ceph OSD

2.Deploy

验证内核版本

# lsb_release -a
$ uname -r

部署client节点ceph

# install ceph
$ ceph-deploy install ceph-client

# ceph-deploy 复制配置文件
$ ceph-deploy admin ceph-client

# 注意权限
$ chmod +r /etc/ceph/ceph.client.admin.keyring

3.Create a block device pool

  1. deploy节点创建池，名称建议使用"rbd"
  2. 块设备Command，Examples

少于5个OSD设置pg_num为128
5到10个OSD设置pg_num为512
10到50个OSD设置pg_num为1024
大于50OSD，REFERNECE

# 查看集群池 
$ ceph osd lspools

# deploy节点创建池
$ ceph osd pool create rbd 128 
# deploy节点初始化池 
$ rbd pool init rbd

# ceph-client 创建块设备镜像
# e.g: rbd create --size {megabytes} {pool-name}/{image-name}
$ rbd create rbd1 --size 1G --image-feature layering --name client.admin
# 镜像映射到设备
$ rbd map --image rbd1 --name client.admin
# 创建文件系统来使用块设备
$ mkfs.ext4 -m0 /dev/rbd/rbd/rbd1
# ceph-client节点挂载文件系统
$ mkdir /data/ceph-block-device
# mount /dev/rbd/{pool-name}/foo /mnt/ceph-block-device
$ mount /dev/rbd/rbd/rbd1 /data/ceph-block-device
$ cd /data/ceph-block-device

# delet ceph pool
$ ceph osd pool delete rbd1 rbd1 --yes-i-really-really-mean-it

REFERNECE

ceph-block-device
